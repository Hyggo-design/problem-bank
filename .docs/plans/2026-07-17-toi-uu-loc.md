# Tối ưu phần lọc — Build Plan

**What we're building:** Tính sẵn bảng tra "nhánh → hệ gốc" một lần, để đường lọc feed khỏi leo ngược cây phân loại cho từng bài mỗi lần lọc.

**Why:** Giữ lọc nhanh khi kho lớn (hàng chục nghìn bài) — kết quả và giao diện y hệt, xuất `.tex` y hệt từng byte.

**Approach:** Thêm một hàm thuần nhỏ `buildRootHeMap(parentMap)` vào util `classification.js` (đã test được), khoá đúng bằng test. `DataGrid` gọi hàm này một lần (memo theo cây phân loại) rồi đọc bảng tra thay cho `getRootHeId` trong vòng lọc. Không đụng xuất `.tex` / CSDL / Rust / giao diện.

**Files we'll create or change:**
- `src/utils/classification.js` — thêm hàm thuần `buildRootHeMap` (dưới `rootHeId` sẵn có)
- `src/utils/classification.test.js` — thêm test cho `buildRootHeMap` (4 ca)

### Task 2: Thêm hàm thuần `buildRootHeMap` + test

**What you'll have when this is done:** Hàm tính sẵn bảng "nhánh → hệ gốc" đã có, và test mới xanh khoá đúng 4 ca (kể cả id mồ côi).

- [ ] Step 1: Thêm hàm vào `src/utils/classification.js`, đặt **ngay dưới** hàm `rootHeId` (dòng ~13), TRƯỚC `groupClassificationByHe`:
      ```js
      // Bảng tra "nhánh -> hệ gốc" tính sẵn MỘT LẦN cho cả cây, để đường lọc
      // khỏi leo ngược parentMap cho từng bài mỗi lần lọc.
      //   parentMap: { [catId]: parent_id }  (mọi id nhánh làm khoá)
      // Trả về: { [catId]: heIdGốc }. Hàm THUẦN — dùng lại `rootHeId` ở trên.
      export const buildRootHeMap = (parentMap) => {
        const map = {};
        for (const id of Object.keys(parentMap)) map[id] = rootHeId(id, parentMap);
        return map;
      };
      ```

- [ ] Step 2: Thêm test vào cuối `src/utils/classification.test.js` (tái dùng `parentMap` fixture đã có ở đầu file):
      ```js
      import { groupClassificationByHe, buildRootHeMap } from './classification';
      // ^ sửa dòng import đầu file cho gồm buildRootHeMap

      describe('buildRootHeMap', () => {
        test('nhánh lá -> hệ gốc', () => {
          const map = buildRootHeMap(parentMap);
          expect(map.ddt).toBe('tc');
          expect(map.snt).toBe('tc');
        });
        test('nhánh giữa -> hệ gốc', () => {
          expect(buildRootHeMap(parentMap).sh).toBe('tc');
        });
        test('hệ gốc -> chính nó', () => {
          expect(buildRootHeMap(parentMap).tc).toBe('tc');
        });
        test('id mồ côi (không có trong cây) -> KHÔNG có khoá', () => {
          const map = buildRootHeMap(parentMap);
          expect(map.khong_ton_tai).toBeUndefined();
        });
      });
      ```

- [ ] Step 3: Chạy đúng tệp test này
      Run: `npm test -- --run src/utils/classification.test.js`
      You should see: các test cũ + 4 test mới **PASS**.

- [ ] Step 4: Lưu tiến độ
      Run: `git checkout -- src-tauri/Cargo.toml 2>/dev/null; git add src/utils/classification.js src/utils/classification.test.js && git commit -m "feat(loc): them buildRootHeMap (bang tra nhanh->he goc) + test"`
