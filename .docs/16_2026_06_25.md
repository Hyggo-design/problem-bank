## 3. Công Việc Đã Thực Hiện (5 task + tinh chỉnh)

### Task 1 — Lệnh Rust `ensure_dir` ✅ (`0aea7a25`)
`lib.rs`: thêm `ensure_dir(path)` (`std::fs::create_dir_all`, để tạo thư mục DB trên ổ D). **Bỏ `get_db_path`** (trỏ ổ C, nay vô nghĩa) + **bỏ `use tauri::Manager`** (hết chỗ dùng). `cargo check` sạch.

### Task 2 — `db.js` nạp DB từ ổ D ✅ (`963d8cd8`)
Trước `Database.load`: lấy `folder = localStorage['pb-db-folder'] || mặc định`; gọi `ensure_dir(folder)`; nạp `sqlite:<folder>\problem_bank.db`; lưu `localStorage['pb-db-path-active']`. **Lỗi (không có ổ D) → fallback** `sqlite:problem_bank.db` (ổ C) để app không chết. Phần tạo bảng/migration/seed giữ nguyên (chỉ đổi *nơi* file, không đổi *cấu trúc/nội dung*).

### Task 3 — Cài đặt hiển thị đường dẫn DB mới ✅ (`e458cb84`)
`SettingsPage`: hiển thị `pb-db-path-active` (thay lời gọi `get_db_path` đã gỡ); Sao lưu/Mở thư mục trỏ theo.

### Task 4 — Danh tính app ✅ (`90f04ded`)
`tauri.conf.json`: identifier `com.hyggo.problembank`, version `1.0.0`, cửa sổ `1200×800` (giữ productName "Problems Bank" + title "Ngân hàng câu hỏi").

### Tinh chỉnh — DB vào thư mục dự án ✅ (`d3b06da9`)
Sau khi Thầy nghiệm thu bản đầu (DB ở `D:\ProblemBank`), Thầy muốn gom vào thư mục dự án → đổi mặc định sang **`D:\0. Problems Bank\app-data`** (db.js + Settings) + **gitignore `/app-data/`**.
