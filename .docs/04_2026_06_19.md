### Ghi chú kỹ thuật cho Task 11 (gợi ý sẵn)
```js
// trong ClassificationPicker, từ categoryIds tính các hệ đang chạm tới:
const parentMap = Object.fromEntries(categories.map(c => [c.id, c.parent_id]));
const heIds = [...new Set(categoryIds.map(id => getRootHeId(id, parentMap)))];
// mỗi heId -> 1 dropdown từ difficulties.filter(d => d.he_id === heId)
```
